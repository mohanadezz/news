# Flutter News App

## Description
This Flutter News App is a feature-rich application designed to provide users with the latest news articles from various sources, ensuring they stay informed with real-time updates.

## Features
- User authentication to secure access to news feeds.
- Customized news feeds based on user interest and preferences.
- Offline reading capability, allowing users to save articles for later.
- Push notifications for the latest news updates.
- User-friendly interface with smooth navigation.
- Share articles on social media platforms.

## Technologies Used
- **Flutter**: A framework for building natively compiled applications for mobile, web, and desktop from a single codebase.
- **Dart**: Programming language used to build the application.
- **Firebase**: Used for user authentication and real-time database services.
- **REST APIs**: For fetching news articles from various sources.
- **Provider**: State management solution for handling app state efficiently.

## Architecture
The app follows a Model-View-ViewModel (MVVM) architecture, which separates the business logic, data, and UI layer:
- **Model**: Represents the data structure of news articles.
- **View**: User interface elements that display data and trigger actions.
- **ViewModel**: Handles the business logic and connects the Model and View.
