## Installation
1. Step 1: Clone the repo
   ```bash
   git clone https://github.com/mohanadezz/news.git
   ```
2. Step 2: Install dependencies
   ```bash
   npm install
   ```

## Usage
To run the project, use the following command:
```bash
npm start
```
